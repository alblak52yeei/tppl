# Интерпретатор языка COW

Интерпретатор эзотерического языка программирования COW, реализованный на Go.

## Быстрый старт

### 1. Сборка проекта
```bash
go build -o cow
```

### 2. Запуск программы COW
```bash
./cow examples/hello.cow
```

### 3. Запуск тестов
```bash
go test -v
```

### 4. Проверка покрытия тестами
```bash
go test -cover
```

## Операторы языка COW

| Оператор | Описание |
|----------|----------|
| `MoO` | Увеличить значение текущей ячейки на 1 |
| `MOo` | Уменьшить значение текущей ячейки на 1 |
| `moO` | Перейти к следующей ячейке памяти |
| `mOo` | Перейти к предыдущей ячейке памяти |
| `moo` | Начало цикла (пока значение ячейки != 0) |
| `MOO` | Конец цикла |
| `OOM` | Вывести значение текущей ячейки |
| `oom` | Ввести значение в текущую ячейку |
| `mOO` | Выполнить инструкцию с номером из текущей ячейки |
| `Moo` | Если ячейка = 0, то ввести, иначе вывести |
| `OOO` | Обнулить значение в ячейке |

## Примеры программ

Все примеры находятся в папке `examples/`:

- `hello.cow` - простой вывод числа
- `loop.cow` - пример цикла
- `input_output.cow` - ввод и вывод
- `zero.cow` - обнуление ячейки
- `multiple_cells.cow` - работа с несколькими ячейками
- `nested_loop.cow` - вложенные циклы

## Структура проекта

- `interpreter.go` - основная реализация интерпретатора
- `main.go` - точка входа программы
- `interpreter_test.go` - тесты (34 тест-кейса)
- `examples/` - примеры программ на COW
